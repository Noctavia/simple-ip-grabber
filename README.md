# 🌐 Deadhook
<h2 align="center">⚡ Simple récupérateur d'adresse IP front-end réalisé en JavaScript et HTML !</h3>
<h4 align="center">🌟 Si vous aimez cela, ajoutez une étoile à ce dépôt et laissez un commentaire ! Merci 😊</h3>
<br />

**⚠ AVERTISSEMENT :** Ceci est UNIQUEMENT à des fins éducatives.

<h2 align="left">- Fonctionnalités -</h3>

* ​​`Extrêmement rapide !`
* `Intégration soignée qui contient l'adresse IP, le code postal, le pays, la région et la ville de la victime.`
* `Front-end donc aucun back-end n'est requis. Facile à héberger !`

# Utilisation
- **Étape 1 :** Remplacez `Remplacez ceci par votre webhook` par votre webhook ! (Duh)
- **Étape 2 :** Enregistrez-le et hébergez-le quelque part. (N'importe quel service d'hébergement fera l'affaire !)
- **Étape 3 :** faites en sorte que quelqu'un clique dessus ! Le message apparaîtra instantanément dans votre webhook.
